ℹ️ How it works
  • Python backend (Flask + requests): performs real HTTP requests with no CORS limitation, reads 2xx/4xx/5xx codes, follows redirects and inspects the HTML to detect parking pages.
  • DNS resolution: each domain is resolved to its IPv4 address; both fields are visible in the table and the exported CSV.
  • VirusTotal: optional integration. Verdicts: ☠ MALICIOUS 2+ engines flag the domain · ⚠ SUSPICIOUS 1 malicious or 2+ suspicious or very negative reputation · ✓ CLEAN no threats · ? UNKNOWN no VT data. Free quota = 4 req/min, 500/day. With a 24h cache repeats do not consume quota.
  • Statuses: Active 2xx/3xx with real content · Parked domain for sale / placeholder · 4xx · 5xx · Unreachable DNS / timeout / SSL.
  • Auto-recheck every N hours (4 by default) runs in a background thread. It keeps working even if the browser is closed. State is persisted to SQLite (monitor.db).
  • Inactive → active transitions: when a domain switches from 4xx / 5xx / UNREACHABLE / PARKED to ACTIVE it is tagged NEW and appears in the highlighted alert.
